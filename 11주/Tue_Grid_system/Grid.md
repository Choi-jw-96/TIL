# Grid system

## 개요

### Bootstrap Grid system
웹 페이지의 레이아웃을 조정하는 데 사용되는 12개의 칼럼으로 구성된 시스템

다양항 기기에 크기를 맞추기 위해서 적당히 크고 약수가 많은 12로 나눈다

## [Grid system 클래스와 기본 구조](Grid.html)

### 핵심 클래스
```
container
    row
        col(4) col(4) col(4)
```
1개의 row안에 12개의 column으로 구성 각 요소는 몇 칸을 차지할 것인가?

#### basic
#### nesting
#### offset
#### Gutter
