#CLI
## 정의
``` 
커맨드 인터페이스(= 명령어 인터페이스)
가상터미널이나 텍스트 너미널을 통해 컴퓨터와 상호작용하는 방식

ex. command.com, cmd...

=> 명령어 기반의 인터페이스
```


## GRI와 차이?
* GRI : 그래픽 기반의 인터페이스
        폴더 등 우리가 보는 컴퓨터 화면


* CLI : 명령어 기반의 인터페이스
        삭제시 휴지통에 들어가지 않는다.
        cmd처럼 컴퓨터 언어로 입력


## 주의사항
우측 상단의 ``폴더`` 확인 필수!(습관 필요)

(~ 폴더명)

## 간단한 사용 문법
- ls : 목록 보기, 열기
- mkdis : 폴더 만들기

-cd : 폴더 이동
    
    cd .. : 이전 폴더
