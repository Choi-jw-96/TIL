## [그래프](Graph.md)
```
정점과 이를 연결하는 간선들의 집합(비선형)

정점 : 간선으로 연결되는 노드(역)

간선 : 정점간의 관계 선 (기찻길)

인접 : 두개의 정점이 하나의 간선으로 직접연결 된 상태 

차수 : 하나의 정점에 연결된 간선 개수

종류
    1. 무방향 그래프
    : 간선의 방향이 없음

    2. 유방향 그래프
    : 간선의 방향이 있음

표현방식
    1. 인접 행렬
    : 0, 1로 표현하는 행렬 방식
      간선 대비 정점이 적을때 사용

    2. 인접 리스트
    : 리스트로 인접 정접을   순차적으로 표현하는 방식
      정점 대비 간선이 적을때 사용
```
